<p>
In a three-body decay \(a \to b\, c\, d\), the kinematics are completely described by two independent Lorentz invariants. A natural choice is the pair of Mandelstam-like variables:
</p>

$$
s = (p_c + p_d)^2, \qquad t = (p_b + p_d)^2
$$

<p>
The <strong>Dalitz plot</strong> — introduced by R.H. Dalitz in 1953 — is a scatter plot of event coordinates \((s, t)\). Because the two-body phase space integration over the third invariant is trivial, the doubly differential decay rate takes the remarkably clean form:
</p>

$$
\tau_a \frac{d\Gamma}{ds\, dt}(a \to b\, c\, d) = \frac{1}{(2\pi)^3 \, 8 m_a^2} \, |\mathcal{M}|^2(s,t)
$$

<p>
The crucial consequence: <strong>the density of events in the Dalitz plot is directly proportional to \(|\mathcal{M}|^2\)</strong>. Flat density → constant matrix element. A gradient → momentum-dependent coupling. A sharp band → a resonance propagator.
</p>

<p>
This makes the Dalitz plot one of the most powerful diagnostic tools in particle physics: you can read off the structure of the amplitude by eye, before writing a single Feynman diagram.
</p>

---

<h3>The Kinematic Boundary</h3>

<p>
Not all \((s,t)\) pairs are kinematically accessible. The physical region — the interior of the Dalitz plot — is bounded by the condition that all three final-state momenta are real and positive. For given masses \(m_a, m_b, m_c, m_d\), this boundary is a closed curve in the \((s,t)\) plane defined by:
</p>

$$
\lambda(s,\, m_a^2,\, t_{\min/\max}^{}) = 0
$$

<p>where \(\lambda(x,y,z) = x^2 + y^2 + z^2 - 2xy - 2yz - 2xz\) is the Källén function. The shape and size of this allowed region itself encodes the masses — which is why the third example below shows a dramatically shrunken plot when \(m_b\) is increased.</p>

---

<script>
(function(){
  /* ── physics parameters ─────────────────────────────── */
  var MA=0.500, MB0=0.0, MB2=0.135, MC=0.140, MD=0.140;
  var MA2=MA*MA, MC2=MC*MC, MD2=MD*MD;

  /* Källén function */
  function kallen(x,y,z){ return x*x+y*y+z*z-2*x*y-2*y*z-2*x*z; }

  /* Exact t limits */
  function tLimits(s,ma,mb,mc,md){
    var ma2=ma*ma, mb2=mb*mb, mc2=mc*mc, md2=md*md;
    var lam_ab=kallen(ma2,mb2,s);
    var lam_cd=kallen(s,mc2,md2);
    if(lam_ab<0 || lam_cd<0) return null;
    
    var E_c_cms = (s + mc2 - md2) / (2 * Math.sqrt(s));
    var E_b_cms = (ma2 - mb2 - s) / (2 * Math.sqrt(s));
    var p_c_cms = Math.sqrt(lam_cd) / (2 * Math.sqrt(s));
    var p_b_cms = Math.sqrt(lam_ab) / (2 * Math.sqrt(s));
    
    var t_min = Math.pow(E_c_cms + E_b_cms, 2) - Math.pow(p_c_cms + p_b_cms, 2);
    var t_max = Math.pow(E_c_cms + E_b_cms, 2) - Math.pow(p_c_cms - p_b_cms, 2);
    
    return [t_min, t_max];
  }

  function sRange(ma,mb,mc,md){
    return [(mc+md)*(mc+md), (ma-mb)*(ma-mb)];
  }

  function M2_ex1(s,t){ return (MA2-s)*(MA2-s); }
  function M2_ex2(s,t){ return Math.pow(s+2*t-MA2-2*MC2, 2); }
  function M2_ex3(s,t){
    var MB2_=MB2*MB2, MD2_=MD*MD;
    return Math.pow(s+2*t-MA2-2*MB2_-MD2_, 2);
  }

  var examples=[
    {mb:MB0, m2:M2_ex1},
    {mb:MB0, m2:M2_ex2},
    {mb:MB2, m2:M2_ex3}
  ];

  function drawExample(canvasId, ex){
    var canvas=document.getElementById(canvasId);
    if(!canvas) return;
    
    var W = canvas.clientWidth || 680;
    if(W === 0) W = 680;
    var H = Math.round(W * 0.64);

    var dpr = window.devicePixelRatio || 1;
    canvas.width = W * dpr;
    canvas.height = H * dpr;
    
    var ctx = canvas.getContext('2d');
    ctx.scale(dpr, dpr);

    var isDark = document.documentElement.getAttribute('data-theme')==='dark';
    var bgC   = isDark ? '#1a1a1a' : '#fafafa';
    var textC = isDark ? 'rgba(255,255,255,0.8)' : 'rgba(0,0,0,0.8)';
    var gridC = isDark ? 'rgba(255,255,255,0.08)' : 'rgba(0,0,0,0.06)';
    var axisC = isDark ? 'rgba(255,255,255,0.3)' : 'rgba(0,0,0,0.3)';

    ctx.fillStyle = bgC;
    ctx.fillRect(0, 0, W, H);

    // محدوده محورها مطابق با PDF برای ایجاد فضای طبیعی در سمت راست
    var sA = 0.07, sB = 0.26; 
    var tA = 0.01, tB = 0.14; 

    var pad = {l: 60, r: 90, t: 20, b: 50};
    var pw = W - pad.l - pad.r;
    var ph = H - pad.t - pad.b;

    if(pw <= 0 || ph <= 0) return;

    function sx(s){ return pad.l + (s - sA) / (sB - sA) * pw; }
    function ty(t){ return pad.t + ph - (t - tA) / (tB - tA) * ph; }

    var imgW = Math.round(pw * dpr);
    var imgH = Math.round(ph * dpr);
    var img = ctx.createImageData(imgW, imgH);

    var sr = sRange(MA, ex.mb, MC, MD);
    var sMin = sr[0], sMax = sr[1];

    var m2Max = 0;
    for(var iy=0; iy<imgH; iy++){
      var t_ = tA + (1 - (iy + 0.5) / imgH) * (tB - tA);
      for(var ix=0; ix<imgW; ix++){
        var s_ = sA + (ix + 0.5) / imgW * (sB - sA);
        if(s_ > sMax) continue; 
        var lim = tLimits(s_, MA, ex.mb, MC, MD);
        if(!lim) continue;
        if(t_ < lim[0] || t_ > lim[1]) continue;
        var v = ex.m2(s_, t_); if(v > m2Max) m2Max = v;
      }
    }

    for(var iy=0; iy<imgH; iy++){
      var t_ = tA + (1 - (iy + 0.5) / imgH) * (tB - tA);
      for(var ix=0; ix<imgW; ix++){
        var s_ = sA + (ix + 0.5) / imgW * (sB - sA);
        var idx = 4 * (iy * imgW + ix);
        
        var lim = tLimits(s_, MA, ex.mb, MC, MD);
        var inBounds = s_ <= sMax && lim && (t_ >= lim[0] && t_ <= lim[1]);
        
        if(!inBounds){
          img.data[idx]   = isDark ? 26 : 250;
          img.data[idx+1] = isDark ? 26 : 250;
          img.data[idx+2] = isDark ? 26 : 250;
          img.data[idx+3] = 255;
        } else {
          var frac = m2Max > 0 ? Math.min(ex.m2(s_, t_) / m2Max, 1) : 0;
          var shade = Math.round(isDark ? (frac * 225) : ((1 - frac) * 230));
          img.data[idx]   = shade;
          img.data[idx+1] = shade;
          img.data[idx+2] = shade;
          img.data[idx+3] = 255;
        }
      }
    }

    var tempCanvas = document.createElement('canvas');
    tempCanvas.width = imgW;
    tempCanvas.height = imgH;
    tempCanvas.getContext('2d').putImageData(img, 0, 0);

    ctx.drawImage(tempCanvas, 0, 0, imgW, imgH, pad.l, pad.t, pw, ph);

    /* ── Drawing the boundary line ── */
    ctx.beginPath();
    var first = true;
    for(var i=0; i<=350; i++){
      var frac = i / 350;
      var s_ = sMin + (sMax - sMin) * frac;
      var lim = tLimits(s_, MA, ex.mb, MC, MD);
      if(!lim) continue;
      if(first){ ctx.moveTo(sx(s_), ty(lim[1])); first = false; }
      else ctx.lineTo(sx(s_), ty(lim[1]));
    }
    for(var i=350; i>=0; i--){
      var frac = i / 350;
      var s_ = sMin + (sMax - sMin) * frac;
      var lim = tLimits(s_, MA, ex.mb, MC, MD);
      if(!lim) continue;
      ctx.lineTo(sx(s_), ty(lim[0]));
    }
    ctx.closePath();
    ctx.strokeStyle = isDark ? 'rgba(255,255,255,0.7)' : 'rgba(0,0,0,0.75)';
    ctx.lineWidth = 1.5;
    ctx.stroke();

    /* ── Drawing grids ── */
    ctx.strokeStyle = gridC;
    ctx.lineWidth = 0.6;
    var xTicks = [0.10, 0.15, 0.20, 0.25];
    var yTicks = [0.02, 0.04, 0.06, 0.08, 0.10, 0.12];

    xTicks.forEach(function(v){
      ctx.beginPath(); ctx.moveTo(sx(v), pad.t); ctx.lineTo(sx(v), pad.t+ph); ctx.stroke();
    });
    yTicks.forEach(function(v){
      ctx.beginPath(); ctx.moveTo(pad.l, ty(v)); ctx.lineTo(pad.l+pw, ty(v)); ctx.stroke();
    });

    /* ── Axis labels and ticks ── */
    ctx.fillStyle = textC;
    ctx.font = '12px serif';
    ctx.textAlign = 'center';
    xTicks.forEach(function(v){
      ctx.fillText(v.toFixed(2), sx(v), pad.t + ph + 18);
    });
    yTicks.forEach(function(v){
      ctx.textAlign = 'right';
      ctx.fillText(v.toFixed(2), pad.l - 8, ty(v) + 4);
    });

    ctx.fillStyle = textC;
    ctx.font = 'bold 14px serif';
    ctx.textAlign = 'center';
    ctx.fillText('s (GeV²)', pad.l + pw/2, H - 12);

    ctx.save();
    ctx.translate(16, pad.t + ph/2);
    ctx.rotate(-Math.PI/2);
    ctx.fillText('t (GeV²)', 0, 0);
    ctx.restore();

    /* Color bar */
    var cbX = W - pad.r + 28, cbW = 14, cbH = ph;
    var grad = ctx.createLinearGradient(0, pad.t, 0, pad.t + ph);
    if(isDark){
      grad.addColorStop(0, 'rgb(225,225,225)');
      grad.addColorStop(1, 'rgb(26,26,26)');
    } else {
      grad.addColorStop(0, 'rgb(0,0,0)');
      grad.addColorStop(1, 'rgb(235,235,235)');
    }
    ctx.fillStyle = grad;
    ctx.fillRect(cbX, pad.t, cbW, cbH);
    ctx.strokeStyle = axisC;
    ctx.lineWidth = 0.8;
    ctx.strokeRect(cbX, pad.t, cbW, cbH);

    ctx.fillStyle = textC;
    ctx.font = '11px serif';
    ctx.textAlign = 'left';
    ctx.fillText('high', cbX + cbW + 6, pad.t + 10);
    ctx.fillText('low', cbX + cbW + 6, pad.t + cbH - 2);

    ctx.save();
    ctx.translate(cbX + cbW + 42, pad.t + cbH/2);
    ctx.rotate(Math.PI/2);
    ctx.font = '11px serif';
    ctx.textAlign = 'center';
    ctx.fillText('τ_a dΓ/ds dt', 0, 0);
    ctx.restore();
  }

  var panels=['dtz-p0','dtz-p1','dtz-p2'];
  var tabs=document.querySelectorAll('.dtz-tab');

  window.dtzShow=function(i){
    panels.forEach(function(id,j){
      document.getElementById(id).classList.toggle('active',j===i);
      tabs[j].classList.toggle('active',j===i);
    });
    setTimeout(function(){ drawExample('dtz-c'+i, examples[i]); }, 40);
  };

  function drawAll(){
    var activeIdx = 0;
    tabs.forEach(function(tab, i){ if(tab.classList.contains('active')) activeIdx = i; });
    drawExample('dtz-c'+activeIdx, examples[activeIdx]);
  }

  window.addEventListener('resize', drawAll);
  document.addEventListener('DOMContentLoaded', function(){ setTimeout(drawAll, 100); });
  setTimeout(drawAll, 200);
})();
</script>

<h3>What the Plots Are Telling Us</h3>

<p>
<strong>Example #1</strong> — The matrix element \(|\mathcal{M}|^2 \propto (m_a^2 - s)^2\) depends only on \(s\). Accordingly, the density contours are horizontal lines: for fixed \(s\), the rate is the same regardless of \(t\). The distribution peaks at the smallest accessible \(s = (m_c+m_d)^2\), the threshold for the \(c\)–\(d\) pair.
</p>

<p>
<strong>Example #2</strong> — Here \(|\mathcal{M}|^2 \propto (s + 2t - m_a^2 - 2m_c^2)^2\). The argument is a linear combination of both invariants, so the zero of \(\mathcal{M}\) forms a straight line in the \((s,t)\) plane — a <em>nodal line</em> — and the density contours are parallel to it. Events pile up in the corners farthest from the node.
</p>

<p>
<strong>Example #3</strong> — Same gradient structure as Example #2, but with \(m_b = 135\,\text{MeV} \approx m_\pi\). The non-zero mass of the third particle shifts the threshold and shrinks the phase space region dramatically, leaving only a compact island in the lower-left of the \((s,t)\) plane. The lesson: <em>phase space geometry encodes the masses</em>, while density variations within the boundary encode \(|\mathcal{M}|^2\).
</p>

<h3>A Practical Reading Guide</h3>

<p>When confronted with an unknown Dalitz plot, the following questions guide the analysis:</p>

<ol>
  <li><strong>Is the density uniform?</strong> If yes, \(|\mathcal{M}|^2 = \text{const}\) and the decay is governed by pure phase space.</li>
  <li><strong>Are the contours parallel to one axis?</strong> Then \(|\mathcal{M}|^2\) depends only on the other invariant.</li>
  <li><strong>Are there bands of high density crossing the plot?</strong> These are resonances: an intermediate state \(a \to R\, b \to c\, d\, b\) with propagator \(\sim 1/(s - m_R^2)\) produces a horizontal band at \(s = m_R^2\).</li>
  <li><strong>Are there empty regions or nodes?</strong> These indicate zeros of the amplitude, often from angular momentum selection rules or destructive interference.</li>
  <li><strong>What is the shape of the boundary?</strong> This directly constrains the final-state masses through the Källén function.</li>
</ol>
